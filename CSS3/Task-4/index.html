Color & Background Styling

HEX Colors

Hexadecimal color values use # followed by 6 digits (0-9, A-F).

This box uses HEX color: #ecf0f1 background with #2c3e50 text

Primary: #3498db

Success: #27ae60

Danger: #e74c3c

RGBA Colors (with Transparency)

RGBA values include an alpha channel (0-1) for transparency control.

This box uses RGBA: rgba(52, 152, 219, 0.2) with transparent background

Format: rgba(red, green, blue, alpha)

Alpha: 0 = fully transparent, 1 = fully opaque

Linear Gradient

Linear gradients blend colors in a straight line across the element.

Linear Gradient: 90deg | Blue → Green → Red

Syntax: background: linear-gradient(angle, color1, color2, ...);

Radial Gradient

Radial gradients blend colors from center outward in a circular or elliptical pattern.

Radial Gradient: Circle | Blue → Dark Blue

Syntax: background: radial-gradient(shape, color1, color2, ...);

Diagonal Gradient

Gradients can be positioned at any angle using degrees (0-360).

Diagonal Gradient: 135deg | Purple → Pink

Common Angles: 0deg (top), 90deg (right), 180deg (bottom), 270deg (left)

Background Image with Fallback

Combine background images with color overlays for better text readability.

Background: Image overlay + Gradient + Fallback Color

Note: Body has gradient background with pattern fallback
