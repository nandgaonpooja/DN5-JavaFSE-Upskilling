Link and List Styling

Link Pseudo-classes

Links have different states that can be styled individually:

:link - Unvisited link (Blue)
:visited - Visited link (Purple)
Hover over link - :hover state (Red with underline)
Click a link - :active state (Orange)

Best Practices

  • Always provide visual feedback for :hover state
  • Ensure :visited links are distinguishable
  • Use consistent colors across links
  • Remove default list styling when creating custom lists
  • Add proper spacing with margin and padding
  • Test keyboard navigation (Tab key)
  • Ensure sufficient color contrast for readability
