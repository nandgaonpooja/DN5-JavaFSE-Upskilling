Master CSS Selectors

Learn how to select and style HTML elements

1. Universal Selector (*)

The universal selector targets all elements on the page. Used for resetting default browser styles.

Applied: Margin and padding reset for all elements

2. Element Selectors

Element selectors target all instances of a specific HTML tag.

Applied: All paragraphs have color #555 and line-height 1.6

3. ID Selectors (#)

ID selectors target a unique element by its ID attribute. Each ID should be unique on a page.

Applied: #mainHeader has gradient background and custom styling

4. Class Selectors (.)

Class selectors target elements with a specific class. Classes can be reused across multiple elements.

5. Grouping Selectors (,)

Group multiple selectors to apply the same styles. Reduces code duplication.

Applied: .eventCard h3 and .eventCard p have margin: 0

6. Attribute Selectors

Target elements based on their attributes.

7. Pseudo-class Selectors

Target elements in specific states.

  • Hover me to see :hover state
  • Visited link shows :visited state
  • Active link shows :active state

8. Child Selectors (>)

Target direct children of an element. All h2 elements that are direct children of section get bold weight.
