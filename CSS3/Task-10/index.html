Responsive Design Breakpoints

Desktop (≥1200px): Full 3-column grid, horizontal navigation menu, large font sizes

Tablet (768px - 1199px): 2-column grid, adjusted spacing, slightly smaller fonts

Mobile (< 768px): Single column layout, vertical navigation menu, optimized for touch

Small Mobile (< 480px): Extra small fonts and spacing for very small screens

Flexible Units Used

Percentages (%): Relative to parent element - used for responsive widths

Viewport Width (vw): Relative to viewport - useful for responsive font sizing

Viewport Height (vh): Relative to viewport height - used for full-height sections

Flexible Grid (fr): Distributes available space in CSS Grid for flexible layouts

clamp(): Sets minimum, preferred, and maximum values for fluid sizing

Mobile-First Approach

This design uses a mobile-first approach, meaning:

1. Base styles target mobile devices (smallest screens)

2. Media queries add enhancements for larger screens

3. Progressive enhancement ensures functionality on all devices

This approach is more efficient and performant for mobile users who have limited bandwidth.

Navigation Behavior

Desktop & Tablet: Navigation links displayed horizontally in a row for quick access

Mobile: Navigation links stack vertically, taking up full width for easy touch targets

The mobile navigation is easier to tap and doesn't use extra horizontal space on small screens.

Card Grid Responsiveness

Desktop: 3 cards per row (grid-template-columns: repeat(3, 1fr))

Tablet: 2 cards per row (grid-template-columns: repeat(2, 1fr))

Mobile: 1 card per row (grid-template-columns: 1fr)

Cards automatically adjust width and stacking based on screen size.

Font Size Scaling

Fixed sizes: Headers use fixed sizes that get smaller on mobile

Responsive sizes: Body text uses percentage or em-based sizing to scale with viewport

Fluid typography: Use clamp() for fonts that scale smoothly between min and max sizes

Example: font-size: clamp(1rem, 2.5vw, 2rem) scales fluidly between 1rem and 2rem

DevTools Testing Tips

Chrome DevTools Device Mode: Press Ctrl+Shift+M to toggle device toolbar

Preset Devices: Choose from iPhone, iPad, Android devices for accurate preview

Custom Dimensions: Test any screen size with custom width/height inputs

Responsive Mode: Drag the edge to continuously resize and see real-time changes

Orientation Toggle: Test portrait and landscape layouts
