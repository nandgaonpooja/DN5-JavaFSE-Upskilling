Box Model & Layout Control

Understanding the Box Model

Every element in CSS consists of: Content → Padding → Border → Margin

Content

The actual content (text, images)

Padding

Space inside the border

Border

Outline around padding

Margin

Space outside the border

Border Styles

Solid Border:

Solid border style - most common

Dashed Border:

Dashed border creates a broken line pattern

Dotted Border:

Dotted border uses dots instead of lines

Double Border:

Double border creates two parallel lines

Custom Borders (different sides):

Top: solid blue | Right: dashed red | Bottom: solid green | Left: dotted orange

Outline vs Border

Border: Takes up space in the layout

Outline: Does NOT take up space, drawn outside border

DevTools Tips

  • Open Chrome DevTools (F12)
  • Right-click any element and select "Inspect"
  • In the Styles panel, see the box model diagram
  • Hover over the visual box model to highlight different parts
  • Click values to edit them in real-time
  • Test different margin/padding values interactively
  • See computed styles tab for final calculated values
