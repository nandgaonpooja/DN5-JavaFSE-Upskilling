CSS Inclusion Methods Demo

Inline Internal External

Inline CSS

This heading has an inline style applied directly: style="color: red;"

Inline styles are applied directly to individual elements using the style attribute. They have the highest specificity but are not reusable.

Internal CSS (Embedded in <style>)

This section is styled using the <style> tag in the <head> section. The background color and fonts are defined there.

Internal stylesheets are useful for single-page applications but not ideal for multi-page websites.

External CSS (Linked File)

This section and all sections above use an external stylesheet styles.css

External stylesheets are the best practice for most projects. They are reusable across multiple pages, cacheable by browsers, and keep HTML clean.

Advantage: One CSS file for the entire website!
