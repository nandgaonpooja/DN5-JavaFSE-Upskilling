Storage Information

localStorage

Persistent storage that survives browser closure

Used for: Long-term user preferences

sessionStorage

Temporary storage cleared when browser tab closes

Used for: Temporary session data

How It Works

1. Select your preferred event type from the dropdown

2. Click "Save Preference" to store it locally

3. Close and reopen the browser - your preference is remembered!

4. Use "Clear All Preferences" to reset everything

5. Click "View Storage" to see what's stored
